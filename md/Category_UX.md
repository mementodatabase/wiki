This is the category page for the Memento User Interface, or User Experience.

The main screens in Memento are:
**The [Libraries list](Libraries List screen.md)** — Shows a list of libraries within the current group
**The [Entries list](Library Entries List screen.md)** — Shows the entries within the current library
**The [Entry View card](Entry View card.md)** — Shows the fields within the current page of the current entry, with tabs for the other pages
**The [Entry Edit card](Entry Edit card.md)** — Shows the fields of the current page of the current entry prepared for editing, with tabs for the other pages
