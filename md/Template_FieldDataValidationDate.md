After selecting Data Validation, a pop-up will appear for entering validation criteria. For Date & DateTime fields, the input will be deemed **valid if**:

**Today** — the date is today.
**Yesterday** — the date is yesterday.
**This week** — the date is some time last week.
**This month** — the date is some time this month.
**Last week** — the date is some time left week.
**Last month** — the date is some time last month.
**Custom** — the date falls between the two dates entered below.
**Is empty** — the field is empty.
**Is not empty** — the field is not empty.
**This year** — the date is some time this year.
