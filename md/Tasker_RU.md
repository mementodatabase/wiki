---
title: "Tasker RU"
---

Memento Database поддерживает интеграцию с приложениями [Tasker](https://tasker.dinglisch.net/) и [Locale](https://www.twofortyfouram.com/).
Tasker is an application for Android which performs tasks (sets of actions) based on contexts (application, time, date, location, event, gesture) in user-defined profiles or in clickable or timer home screen widgets.

Вы можете создавать/редактировать записи Memento в качестве реакции на определенные события или изменения состояния устройства.
Например вы можете автоматически сохранять в библиотеках входящие смс-сообщения, информацию о звонках, историю местоположений и многое другое.

   1. Настройка задачи создания записи
1. У вас уже должна быть создана библиотека в которую вы будете записывать события из Tasker-а.
1. Перейдите в Tasker и откройте диалог создания новой задачи.
1. Выберите Plugin > Memento Database.
1. Выберите Configuration, а затем выберите библиотеку.
1. Добавьте  значения которые будут записаны в поля новой записи. Вы можете использовать переменные из Tasker-а : http://tasker.dinglisch.net/userguide/en/variables.html

   1. Настройка задачи изменения записи
1. Выполните такие же действия как и для задачи создания записи. 
1. Включите опцию “Редактировать запись”. 
1. Выберите поле которое будет являться ключевым при редактировании записи. Значения из Tasker-а будут записываться в записи, значение ключевого поле которых соответствует указанному значению. В качестве значения ключевого поля можно использовать переменные из Tasker-а. 

Все значения передаваемые в Memento и значение ключевого поля, должны иметь тот же самый формат как и при [импорте из csv](Importing and exporting data.md).

   1. События
Memento может являться источником событий для Tasker. Поддерживаются следующие типы событий: создание записи, модификация записи и удаление записи. 
При обработки события в Tasker-е доступны значения полей записи, являющейся источником события, в виде переменных вида - *%имя_поля*.

Шаги для создания события:
1. Перейдите в Tasker и откройте диалог создания нового события.
1. Выберите Plugin > Memento Database и нужный тип события.
1. Выберите Configuration, а затем выберите библиотеку.

События генерируются только при операциях с записями через интерфейс приложения. События не генерируются если записи изменяются через синхронизацию.
