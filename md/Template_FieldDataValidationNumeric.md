After selecting Data Validation, a pop-up will appear for entering validation criteria. For *numeric fields*, the input will be deemed valid if:
**Less than (default)** — its value is less than the value entered below.
**Less than (default)** — its value is less than the value entered below.
**Greater than** — its value is greater than the value entered below.
**Equal to** — its value is equal to the value entered below.
**Not equal to** — its value is not equal to the value entered below.
**Empty** — the field is empty.
**Not empty** — the field is not empty.
**Between** — its value is between the two numbers entered below.
