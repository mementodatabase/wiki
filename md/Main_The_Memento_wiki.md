1. The Memento Database Wiki
You've arrived at the user-developed-&-maintained wiki for the Memento Database application. After a very brief introduction and some information about getting started with Memento, you'll get to the index into the wiki.

   1. Introduction
You'll see in the banner menu at the top of the page the main sections of the wiki. The Home, Getting started, & wiki support sections are on this page below. The others are links to other pages.

Let's get to it...

   1. Getting started with Memento Database
You can learn about Memento Database in the About Memento page, but all you initially might need to know are...
**What it is** — Memento Database is an app published by LuckyDroid that runs on mobile devices (Android only so far) & "desktop" (often actually laptop) Microsoft Windows, Apple macOS, & Linux computers. The mobile app
**What the parts are** — The mobile edition app can have any number of local *libraries*, which correspond to tables in a relational DBMS. Libraries may be copied up to the Memento Cloud, as well, which allows direct sharing of the libraries with other devices & computers. In the [About Memento](Main:About Memento Database.md) section of the wiki and on the Memento Web site about various aspects of the Cloud, such as cloud subscriptions & their prices & so on. There is also a [forum for Memento users](https://groups.google.com/g/mementodatabase)]. In addition to [this wiki](Main:Memento Database wiki.md), Memento includes product Help within the app.
**How they generally work together** — See [the Memento About page](Main:About Memento Database.md) for more information on Memento Database.

      1. The Memento Database Web site
The official [Memento Database Web site](https://mementodatabase.com) is at https://mementodatabase.com.

      1. Downloading
Each edition and each platform (device or computer & OS) has a Download page. Generally, you'll download the most recently released app. If you are already a user and in the beta testing program, there may be other app images available for download. The mobile edition can be downloaded in the [Google Play store](https://play.google.com/store)

The app is written in the Java programming language, and while the mobile edition has Java embedded, each desktop edition platform has the latest Java Development Kit or Java Runtime Engine (JDK or JRE) available for download. That completes the set of downloads necessary.
