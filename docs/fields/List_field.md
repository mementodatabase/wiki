---
title: "List field"
---

A Dynamic List field is a set of text strings with checkboxes that you can create for each entry.  An example for use would be a daily to do list.

## Advanced Parameters

**Default Value** — Similar to the hint option, the Default value is available in the desktop edition, however, any data input does not generate a visible result.

## Display Options
**Field name** — Display the name of the field in the card entry (By default, On).  By this, it is meant that the field name will or will not be displayed on either the Entry View card or the Entry Edit card.

**Font** — :; Family: Regular, Light, Condensed, Thin, Medium, Monospace, Serif Monospace. By default, Regular.
**Style** — Normal, Bold, Italic, Bold Italic. By default, Normal.
**Size** — By default, 18. Direct entry to change.
**Color** — By default, White. Palette of 20 colors, backed up by a continuously variable color chooser.

**Dependencies** — Ability to add one or more visibility dependencies upon fields with qualifying types.
