---
title: "Cloud Library Status card"
---

This card displays information about the following information about this library as it exists within the Memento Cloud.

**Library address** — Displays the library's cloud URL.

**Information block** — contains:
**Last sync** — The date & time of last sync to the cloud
**Entries** — The current number of entries in the library
**Created** — The date & time of library creation
**Modified** — The date & time of last library modification
**Size** — The account of space taken up by the library
**Files count** — The number of files associated with this library.

**Notification setting** — Sends the user a notification when anyone else modifies the library.

**Add people button** — Allows the user to grant people, one-by-one, permission to access the library.

**Public access button** — Allows the user to grant permission to access the library to the public, if desired.
