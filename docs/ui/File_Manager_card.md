---
title: "File Manager card"
---

The File Manager card is launched by a screen when it needs to identify either a file as a source or a directory as a destination.

## Selecting a file as a source
**A title bar containing** — * the title **Select a file**
- the current directory path
- a button to exit the card

**A list** — of directories and files

**A filename extension filter** — which may be set on or off

Tap directories one by one of/as needed to navigate to the desired file. Then touch the file to select it as the source.

## Selecting a directory as a destination
**A title bar containing** — * The title **Select a directory**
- the current directory path
- a button to exit the card

**A list** — of directories

Tap directories one by one of/as needed to navigate to the desired file. Then touch the OK icon to select it as the destination.
